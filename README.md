# Лабораторная работа №15 “Разработка простого веб-сервиса”

## Цели и задачи

### Цели

1. Освоить основы работы с микрофреймворком **Flask**.
2. Научиться создавать базовый веб‑сервер, обрабатывать запросы и формировать ответы.
3. Закрепить навыки разработки REST‑подобных сервисов.
4. Изучить структуру проекта и работу с маршрутами.

### Задачи

1. Создать простой веб‑сервис на Flask.
2. Реализовать маршруты (routes) для обработки запросов.
3. Принимать данные от пользователя: URL‑параметры, формы, JSON.
4. Подготовить отчёт и репозиторий.
5. Соблюсти структуру проекта и требования.

## Общие задания

### Задание 1

Создать простейший веб‑сервис на Flask.

Структура проекта:

```
task_1/
    app/
        __init__.py
        [routes.py](http://routes.py)
    [main.py](http://main.py)
    [README.md](http://README.md)
```

В [`routes.py`](http://routes.py) реализовать маршруты:

- `/` — приветствие
- `/hello/<name>` — персональное приветствие
- `/square/<int:number>` — квадрат числа

В [`main.py`](http://main.py) настроить запуск приложения.

## Индивидуальные задания

### Вариант 1

1. `GET /sum` — принимает `a`, `b` через query‑параметры, возвращает сумму.
2. `GET /info` — возвращает JSON с полями: версия, автор, дата запуска.
